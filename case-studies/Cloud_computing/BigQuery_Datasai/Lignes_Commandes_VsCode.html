🐍 Étape 0 — Vérifier Python PowerShell
Vérifie que Python est bien installé avant de continuer.
Tu dois voir Python 3.11.x ou supérieur.
python --version
📦 Étape 1 — Installation des dépendances Python PowerShell
Installe les trois librairies nécessaires en une seule commande.
google-cloud-bigquery → connexion et requêtes BigQuery
pandas → afficher les résultats en DataFrame
db-dtypes → convertir les types BigQuery vers Pandas
pip install google-cloud-bigquery pandas db-dtypes
✅ Étape 1b — Vérifier l'installation PowerShell
Vérifie que les trois librairies sont bien installées.
pip show google-cloud-bigquery pandas db-dtypes
🔐 Étape 2 — Authentification Google Cloud PowerShell
Connecte les librairies Python à ton compte Google Cloud.
Important : différent de gcloud auth login — celui-ci est pour les librairies Python.
gcloud auth application-default login --no-browser
🔐 Étape 2b — Si erreur de navigateur Optionnel
Si l'authentification plante avec le navigateur — ouvrir un deuxième terminal et coller la commande longue qui s'affiche dans le premier terminal.
# Dans un DEUXIÈME terminal PowerShell # Coller ici la commande longue affichée dans le premier terminal : # gcloud auth application-default login --remote-bootstrap="https://..."
🚀 Étape 3 — Lancer le script Python Principal
Exécute les 7 requêtes BigQuery sur les 58 millions de trajets NYC Citi Bike.
python bigquery_datasai.py
🛠️ Commandes utiles PowerShell
Vérifications et configuration du projet Google Cloud.
# Vérifier que l'authentification a marché gcloud auth application-default print-access-token # Vérifier le projet actif gcloud config get-value project # Changer de projet si nécessaire gcloud config set project gen-lang-client-0080670010
