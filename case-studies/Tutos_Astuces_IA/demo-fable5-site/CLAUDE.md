## Mise à jour Projet

Mets à jour le CLAUDE.md avec les infos suivantes : ce projet reconstruit une interface web à partir d'une capture d'écran fournie en image. Le code doit
être en HTML/CSS/JavaScript, sans dépendance externe, organisé en fichiers séparés (index.html, style.css, script.js). Priorité à la fidélité visuelle
